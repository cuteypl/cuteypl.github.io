CSS修改滚动条的默认样式

1、知识点

  • &::-webkit-scrollbar------------>/* 整个滚动条 */
  • &::-webkit-scrollbar-thumb------------>/* 滚动条的滚动滑块 */
  • &::-webkit-scrollbar-track------------>/* 滚动条的滑动轨道 */
  • &::-webkit-scrollbar-button------------>/* 滚动条的按钮 */
  • &::-webkit-scrollbar-corner------------>/* 水平垂直滚动条相交的角落 */
  • &::-webkit-scrollbar-track-piece------------>/* 滚动条没有滑块的轨道部分 */

2、补充

  • &::-webkit-scrollbar:horizontal/vertical(默认)------------>/* 滚动条可选定水平或垂直 */
  • &::-webkit-scrollbar-button:decrement/increment------------>/* 滚动条可选定上下按钮 */
  • &::-webkit-scrollbar-button:horizontal/vertical:decrement/increment------------>/* 方向与两头按钮的组合选择 */
  • &::-webkit-resizer------------>/* 某些元素的corner部分的部分样式，如textarea的可拖动按钮 */

3、IE的

  • &::scrollbar-arrow-color------------>/* 滚动条三箭头的颜色 */
  • &::scrollbar-face-color------------>/* 滚动滑块颜色 */
  • &::scrollbar-track-color------------>/* 滚动条轨道、按钮的背景色 */
  • &::scrollbar-shadow-color------------>/* 滚动滑块边框颜色 */

4、代码示例

/*html*/
<div class="container">
    <div class="wrapper"></div>
</div>

/*css*/
.container{
    width: 200px;
    height: 400px;
    margin: 50px auto;
    overflow: auto;
    background: #333;
}
.wrapper{
    width: 300px;
    height: 800px;
    background: linear-gradient(red,yellow);
}
.container::-webkit-scrollbar { /* 整个滚动条的样式 */
    width: 6px;
    height: 6px;
    background: transparent;
}
.container::-webkit-scrollbar-thumb { /* 滚动条里可滑动的方块的样式 */
    border-radius: 4px;
    background: rgba(255, 255, 255, 0.3);
}
.container::-webkit-scrollbar-corner { /* 水平垂直滚动条相交的角落的样式 */
    background: transparent;
} 
.container:hover::-webkit-scrollbar-thumb { /* 鼠标放上去时的可滑动方块的样式 */
    background: hsla(0, 0%, 53%, 0.6);
}
.container:hover::-webkit-scrollbar-track { /* 鼠标放上去时的滑动轨迹的样式 */
    background: hsla(0, 0%, 53%, 0.1);
}
